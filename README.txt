1) Save the files "url.txt" and "windows_chromedriver.exe" in C Drive of the windows.
2) Set the exe as default PDF opener. Settings > Apps > Default Apps > Search for PDF and select "PDF Viewer.exe" as default app.
3) Open any PDF and select the PDF it will automatically upload the document into the website. 

NOTE : When the website is hosted on intranet change the URL link in the "url.txt" file
